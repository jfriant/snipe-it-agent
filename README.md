# snipe-it-collector
Single-script agent to create new Assets in a Snipe-IT database

The API key and base URL for your server need to be in a file called "config.json":

```json
{
    "apiKey": "YOUR_KEY_HERE",
    "baseUrl": "https://yourserver.example.com"
}
```
